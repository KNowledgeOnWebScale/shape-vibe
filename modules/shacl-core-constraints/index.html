shacl-core-constraints

This module contains examples from section 4 of the SHACL specification. Section 4 lists the following 8 categories of constraint types:

  • Value Type
  • Cardinality
  • Value Range
  • String-based
  • Property Pair
  • Logical
  • Shape-based
  • Other
The examples are available separately allowing to check individually, but there is also one example file covering all the constraints.
