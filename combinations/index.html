ShapeViBe - combinations

This part of the benchmark contains modules to test different combinations of constraints. Modules of this part can thus be used to check if a visual notation or a tool covers different combinations or edge cases of constraints.

min-max-values

This module contains combinations of several constraint types which represent different minimum and maximum values which could be easily confused. For example minimum cardinality on a property, minimum value of the property's value or minimum length of the property's value.
